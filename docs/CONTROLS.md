# Controls Reference

## Simulation

| Key | Action |
|---|---|
| `Space` | Pause / Resume simulation |
| `→` (Right Arrow) | Step one tick (while paused) |
| `Escape` | Quit |

## Camera

| Input | Action |
|---|---|
| Right-click + drag | Pan camera |
| Mouse wheel | Zoom in/out |
| Left-click | Select entity |

## Overlays

| Key | Action |
|---|---|
| `F1` | Toggle debug inspector panel |
| `F2` | Toggle population heatmap |
| `F3` | Toggle influence map |
| `F4` | Toggle faction territory overlay |
| `G` | Toggle tile grid |
| `P` | Toggle pathfinding paths |

## Save / Load

| Key | Action |
|---|---|
| `F5` | Quick save |
| `F9` | Quick load |

## Debug Panel (F1)

| Key | Action |
|---|---|
| `Tab` | Cycle debug panel tabs |

### Debug Tabs

- **Inspector** — Selected entity's component data
- **AI** — Utility scores, personality traits, current state
- **Memory** — Known entities, recent events
- **Timeline** — Global event log
- **Factions** — Faction overview and diplomacy
